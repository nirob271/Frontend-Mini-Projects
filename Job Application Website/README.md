# Job Application Form (Web)

This is a simple job application form built using **HTML** and **CSS**.  
I created this project to practice and improve my **frontend development skills**.

## Features

- Clean and minimal design
- Responsive layout
- Form inputs:
  - Name, Email, Address
  - Job Role (dropdown)
  - City, Pincode
  - Date picker
  - CV upload
- Submit button ("Apply Now")

## Technologies Used

- HTML5  
- CSS3

## How to Use

1. Download or clone the project.
2. Open `index.html` in your browser.
3. Fill in the form and explore the layout.
